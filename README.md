# Weather App Interface

A modern, responsive weather application interface built with HTML and Tailwind CSS.

## Features

- Clean and modern user interface
- Responsive design that works on all devices
- Search functionality for cities
- Current weather display with temperature and conditions
- Additional weather details (humidity, wind speed)
- 5-day weather forecast
- Beautiful gradient background
- Glassmorphism design elements

## Setup

1. Clone this repository to your local machine
2. Open the `index.html` file in your web browser

That's it! The project uses Tailwind CSS via CDN, so no additional setup is required.

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Heroicons (for weather icons)

## Preview

The interface includes:
- A search bar for looking up cities
- Current weather display with temperature and conditions
- Weather details cards showing humidity and wind speed
- A 5-day forecast section
- Responsive design that works on mobile and desktop

## Customization

You can customize the interface by modifying the Tailwind classes in the HTML file. The current design uses:
- A blue-to-purple gradient background
- Glassmorphism effects with backdrop blur
- White text with varying opacity levels for hierarchy
- Responsive grid layouts for different screen sizes
